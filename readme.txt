This application was developed using:

Python 3.8
Windows 11
PyCharm IDE
Please make sure that your environment is compatible with these specifications for optimal performance.


How to Use
To start the Graphical User Interface (GUI) and input your health data, run the app.py file with the following command:
python app.py

To perform Exploratory Data Analysis (EDA) and view visualizations, run the main.py file with the following command:
python main.py
